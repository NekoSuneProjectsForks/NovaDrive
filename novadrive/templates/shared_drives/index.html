How Access Works

Invite only means only explicitly added members can open the shared drive.

Request access means signed-in users can ask the owner for access.

Public means signed-in users can view the shared drive even if they are not members.

Only owners can invite members, manage requests, and change visibility. Admins can create the shared drive and control its storage cap.
